# Usage for my suckless programs

## scat (Suckless Cat)
scat /file
## scp (Suckless Copy)
scp /file /file
## sdate (Suckless Date)
sdate
## secho (Suckless Echo)
secho "text"
## sls (Suckless list)
sls.exe /folder (optional)
## smkdir (Suckless Make Directory)
smkdir /folder
## smv (Sucklesss Move)
smv /folder_or_file /new_place
